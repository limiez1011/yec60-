# yec60%

## Description
The yec60% keyboard is based on the ansi 60% layout and has 61 hotswappable sockets.

## Features
- Built in rp2040 MCU
- QMK firmware
- Hot swap sockets

## Flashing procedure
To flash firmware onto the RP2040 there are 2 pads under the "BOOT" text on the silkscreen, you must short those 2 pins to enter bootloader. Once in bootloader a drive should appear on your computer. Copy the .uf2 file from the QMK folder of the repo and paste it into the drive. Once done unplug the USB cable and plug it back in and it should be flashed.
